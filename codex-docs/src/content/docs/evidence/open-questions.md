## High priority

| Question | Why it matters | How to resolve |
|---|---|---|
| Will developers pay for in-client LLM stream/debug traces? | Core differentiator economics | Prototype; measure repeated usage; interview LLM API builders |
| Can Rambo import real Postman/Insomnia/Bruno workspaces with low loss? | Switching depends on this | Run imports on real exported collections and publish loss reports |
| Does closed-source hurt trust versus Bruno/Yaak? | Positioning risk | Buyer interviews; security-review discovery |
| What exact free team cap maximizes adoption/conversion? | Pricing leverage | Instrument workspace size; A/B once launched |
